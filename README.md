# Cinema Booking System (C++)

A simple console-based Cinema Booking System that demonstrates OOP concepts in C++.

## Features
- **Movie Selection**: Choose from a list of available movies and time slots.
- **Dynamic Pricing**: Includes a student discount (45%) validated by ID.
- **ANSI Color Coding**: Uses escape sequences for a colorful terminal UI.
- **OOP Principles**: Implements Inheritance, Overriding, and Encapsulation.
